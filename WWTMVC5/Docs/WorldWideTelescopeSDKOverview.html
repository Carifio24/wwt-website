WorldWide Telescope SDK Overview

Note: This documentation is preliminary and subject to change.

The WorldWide Telescope SDK enables custom web applications and custom data to be developed and used with Microsoft WorldWide Telescope.

There are currently two versions of WorldWide Telescope -- the Windows application and the Web Control. The Windows version has more features (such as the 3-D Solar System) but cannot be customized by the use of scripts, though can use custom data files. The Web Control can be customized with both scripts and data.

Documentation

The SDK documentation is organized into four components:

  • The WorldWide Telescope Web Control Script Reference is for use with the Web Control only, and describes the API set and use of JScript to build custom user interfaces, load custom data, and other types of application.
  • The WorldWide Telescope Data Files Reference document describes the data file types and formats, including studies (single images of an object in space), surveys (images of the entire sky), tours (animated slide-shows) and communities (public or private groups of users).
  • The WorldWide Telescope Data Tools Guide describes the use of two tools: The Study Chopper and Sphere Toaster. These tools make the preparation of image data for WorldWide Telescope relatively easy.
  • The WorldWide Telescope Projection Reference contains background information on the projection systems used by WorldWide Telescope.

Support

To privately send a specific bug, issue, question or comment direct to the WorldWide Telescope team at Microsoft, go to the WorldWide Telescope Support site, and select Issues and Bugs.

To publicly post a comment regarding the SDK, consider adding a comment to an appropriate entry in the WWT Data Blog.
